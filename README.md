# electricity_meter_zrd

Not finished! In the process!

Устройство представляет из себя оптопорт + Zigbee router.

Плата не заказана и не протестированна!!! Это пока только для ознакомления!!!

4. Энергомера-СЕ102М
5. Нева-МТ124
6. Нартис-100
